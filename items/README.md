# items/

**Archive of photos for items that have been listed or sold.**

Photos move here from `photo-inbox/` after a listing has been successfully created. This is permanent storage — nothing new ever starts here.

## Structure

One subfolder per item:

```
items/
├── kitchenaid-mixer/
│   ├── photo-1.jpg
│   └── photo-2.jpg
└── singer-featherweight-221/
    ├── photo-1.jpg
    └── photo-2.jpg
```

## Don't put new items here

If you have photos of something you want to sell, put them in `photo-inbox/` instead — that's where Claude looks for new items to process.
